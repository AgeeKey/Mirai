# Улучшение кода

## Комментарии для функций
1. Функция `calculateSum` - отвечает за вычисление суммы двух чисел.
2. Функция `fetchData` - выполняет запрос к API для получения необходимых данных.

## Примеры использования
- Для функции `calculateSum`:
  ```python
  result = calculateSum(5, 10)
  print(result)
  ```
- Для функции `fetchData`:
  ```python
  data = fetchData('https://api.example.com')
  print(data)
  ```
