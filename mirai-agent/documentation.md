# Документация проекта

## Структура проекта
- **src/**: Исходные файлы
- **tests/**: Тестовые файлы

## Установка
Для установки зависимостей выполните:
```
npm install
```

## Запуск
Чтобы запустить проект, используйте:
```
npm start
```

## Тестирование
Для выполнения тестов используйте:
```
npm test
```

## Вклад
